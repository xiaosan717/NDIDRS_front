# NDIDRS 夜查寝数据报送系统 - 前端
宿舍夜查信息管理平台前端页面，基于Vue开发，部署于Vercel，对接SpringBoot后端公网接口。

## 项目简介
本系统面向高校宿舍管理，支持四类角色登录：学生/宿舍长、辅导员、宿管、管理员，实现夜查记录提交、宿舍信息管理、检查数据统计等功能。

## 技术栈
- 框架：Vue3
- 构建工具：Vite
- HTTP请求：Axios
- UI组件：Element Plus
- 部署平台：Vercel
- 后端支撑：SpringBoot JDK17

## 环境要求
- Node.js 16+ / 18+
- npm / yarn
